# TL-Dump

### Current version: 1.0.0

### Description:

For all those who are tired of looking for methods in the documentation and obfuscated methods.

### Dependencies:
* [TL-Callbacks](https://github.com/TorayLife/mappet-TL-API/blob/master/TL-Callbacks)

### Install:

1) Create `TL-Dump` script on mappet dashboard.
2) Add `TL-Callbacks` script as a libraries.
3) [Copy this code](https://raw.githubusercontent.com/TorayLife/mappet-TL-API/master/TL-Dump/TL-Dump.js) to your `TL-Dump` script.
4) Done!

### How to use:

Type some code inside black input zone and press "Dump" button.
